# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2025-11-30

### Added
- **Shopping Cart:** A fully functional shopping cart that persists across sessions.
- **Cart Page:** A dedicated page for managing cart items, featuring a list view and a "Scan Mode" carousel for easy checkout at the register.
- **Cart AI Tools:** The AI assistant can now add, remove, clear, and view items in your cart (`add_to_cart`, `remove_from_cart`, `clear_cart`, `view_cart`).
- **Cart Integration:** "Add to Cart" buttons added to product detail pages and a cart indicator in the app bar.

### Changed
- **UI Refinements:** Moved OpenRouter authentication to the side menu to declutter the main app bar.
- **Performance:** Improved stability in store availability checking and fixed UI overflow issues on smaller screens.

## [1.1.0] - 2025-11-29

### Added
- **Intelligent Scan Requests:** The AI agent can now contextually request the user to scan a product to gather more information.
- **Advanced Code Scanning:** Expanded scanner capabilities to support in-store QR codes, enabling the extraction of both UPC and SKU data directly from QR labels.

### Changed
- **Scan Flow Overhaul:** Re-engineered the product scanning workflow to provide a smoother, more intuitive user experience.
- **Product Display UI:** Refined the Product Display Page interface with an enhanced design, improving readability and visual presentation of product details.
